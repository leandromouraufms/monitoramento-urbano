# SISMONI - Sistema de Monitoramento Urbano (São Conrado - UFMS)

Este projeto é um protótipo funcional desenvolvido para o **Projeto Integrador II**, focado na fiscalização de terrenos baldios e controle de focos de doenças (como a Dengue) no bairro São Conrado, em Campo Grande/MS.

## 🚀 Funcionalidades
- **Acesso Dual:** Fluxos distintos para Cidadãos (denúncia anônima) e Fiscais (gestão).
- **Painel de Monitoramento:** Dashboard com indicadores de focos críticos e situações em vistoria.
- **Laudo de Vistoria:** Interface para agentes públicos registrarem vistorias técnicas no local.
- **Sistema de Notificações:** Simulação de envio de alertas para proprietários de terrenos.
- **Design Responsivo:** Adaptado para uso em dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas
- [React.js](https://reactjs.org/) - Framework Frontend
- [Vite](https://vitejs.dev/) - Ferramenta de Build
- [Tailwind CSS](https://tailwindcss.com/) - Estilização
- [Lucide React](https://lucide.dev/) - Ícones
